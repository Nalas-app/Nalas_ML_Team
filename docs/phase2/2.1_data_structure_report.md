## 5. Data Relationships

```
menu_items (34)  ←──1:N──→  recipes (586)  ──N:1──→  ingredients (129)
     │                                                       │
     │                                                       │
     └──────── recipe_costs (17) ←── calculated from ────────┘
```

**Key Relationships:**
- Each **menu item** has 1 to 27 **recipe** ingredient rows (avg: 17.2)
- Each **recipe row** references one **ingredient** from the master
- **Recipe costs** are calculated from recipe quantities × ingredient prices
- 225 of 586 recipe ingredient references are **UNMATCHED** (ingredient not found in master)
