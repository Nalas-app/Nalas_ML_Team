## 3. Business Rule Checks

| Rule | Check | Result | Status |
|------|-------|--------|--------|
| All quantities positive | `quantity_per_base_unit > 0` (where not null) | 0 violations | ✅ PASS |
| All costs positive | `price_per_unit >= 0` | 1 zero value (Kadai Masala) | ⚠️ WARNING |
| Profit margin = 10% of subtotal | `profit_margin == subtotal * 0.10` | All 17 correct | ✅ PASS |
| Labor = 15% of ingredient cost | `labor_cost == ingredient_cost * 0.15` | All 17 correct | ✅ PASS |
| Overhead = 10% of ingredient cost | `overhead_cost == ingredient_cost * 0.10` | All 17 correct | ✅ PASS |
| Cost proportional to complexity | Higher ingredient count → higher cost | 2 anomalies (Mutton, Pongal) | ⚠️ WARNING |
