### Step 3: Feature Extraction

```
Raw Data                          Computed Features
────────                          ─────────────────

Recipe + Prices ─────────────▶    ingredient_cost = Σ(qty × price × scaling_factor)
                                  ingredient_count = len(ingredients)
                                  recipe_complexity = weighted_score(ingredient_count, prep_steps)
                                  avg_ingredient_price = mean(prices)
                                  max_ingredient_price = max(prices)

Quantity ─────────────────────▶   scaling_factor = quantity / base_serving_size
                                  quantity_bucket = categorize(quantity)
                                  bulk_discount = lookup_discount(quantity)

Event Date ───────────────────▶   month = event_date.month
                                  season = map_season(month)
                                  seasonal_factor = get_seasonal_adjustment(season)
                                  is_weekend = event_date.weekday() >= 5

Guest Count ──────────────────▶   servings_per_guest = quantity / guest_count
                                  event_scale = categorize_scale(guest_count)
```

## 3. Data Flow — Model Training (Offline)

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│              │     │              │     │              │     │              │
│  Historical  │────▶│  Feature     │────▶│  Model       │────▶│  Evaluate    │
│  Order Data  │     │  Engineering │     │  Training    │     │  & Validate  │
│  + Prediction│     │              │     │              │     │              │
│  Logs        │     │              │     │              │     │              │
└──────────────┘     └──────────────┘     └──────────────┘     └──────┬───────┘
                                                                      │
                                          ┌──────────────┐            │
                                          │              │     Pass?  │
                                          │  Deploy to   │◀── Yes ───┘
                                          │  Production  │            │
                                          │              │     Fail?  │
                                          └──────────────┘     │      │
                                                               ▼      │
                                          ┌──────────────┐            │
                                          │  Investigate │◀── No ────┘
                                          │  & Iterate   │
                                          └──────────────┘

Training Data Pipeline:
─────────────────────────
1. Extract completed orders (with actual costs) from database
2. Join with prediction logs (predicted costs for same orders)
3. Join with recipe data (features at time of prediction)
4. Join with ingredient prices (prices at time of prediction)
5. Generate training features (same as inference pipeline)
6. Split: 70% train, 15% validation, 15% test
7. Train model on training set
8. Evaluate on validation set → tune hyperparameters
9. Final evaluation on test set → report metrics
10. If metrics pass → deploy; else → iterate
```
