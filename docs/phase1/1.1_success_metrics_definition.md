## 1. Key Performance Indicators (KPIs)

### 1.1 Model Performance Metrics

| Metric | Definition | Formula | Target |
|--------|-----------|---------|--------|
| **MAPE** | Mean Absolute Percentage Error | `mean(|actual - predicted| / actual) × 100` | < 10% (production) |
| **MAE** | Mean Absolute Error (₹) | `mean(|actual - predicted|)` | Context-dependent |
| **RMSE** | Root Mean Square Error (₹) | `sqrt(mean((actual - predicted)²))` | < 15% of mean cost |
| **R² Score** | Coefficient of Determination | `1 - SS_res / SS_tot` | > 0.85 |
| **Confidence Accuracy** | % of high-confidence predictions that are accurate | `correct_high_conf / total_high_conf × 100` | > 90% |

### 1.2 Operational Metrics

| Metric | Definition | Target |
|--------|-----------|--------|
| **Prediction Latency (P50)** | Median response time | < 100ms |
| **Prediction Latency (P95)** | 95th percentile response time | < 300ms |
| **Prediction Latency (P99)** | 99th percentile response time | < 500ms |
| **API Availability** | Uptime percentage | > 99% |
| **API Error Rate** | % of requests returning 5xx errors | < 1% |
| **Throughput** | Predictions per second capacity | > 50 req/s |

### 1.3 Business Impact Metrics

| Metric | Definition | Target |
|--------|-----------|--------|
| **Quotation Generation Time** | Time to generate a complete cost quotation | < 5 seconds (entire flow) |
| **Manual Review Rate** | % of predictions flagged for human review | < 20% |
| **Prediction Rejection Rate** | % of predictions overridden by users | < 10% |
| **Cost Accuracy (Post-Order)** | Predicted cost vs actual incurred cost | Within ±10% |

## 3. Measurement Methodology

### 3.1 MAPE Calculation
```python
import numpy as np

def calculate_mape(y_actual, y_predicted):
    """
    Calculate Mean Absolute Percentage Error.
    Excludes cases where actual is zero.
    """
    y_actual = np.array(y_actual)
    y_predicted = np.array(y_predicted)
    
    # Exclude zero actuals
    mask = y_actual != 0
    y_actual = y_actual[mask]
    y_predicted = y_predicted[mask]
    
    mape = np.mean(np.abs((y_actual - y_predicted) / y_actual)) * 100
    return mape
```

### 3.2 Confidence Score Calculation
```
Confidence = f(data_completeness, model_certainty, historical_match)

Where:
- data_completeness: % of required input features available (0-1)
- model_certainty: 1 - prediction_variance / mean_prediction (0-1)
- historical_match: similarity to training distribution (0-1)

Final confidence = weighted_average(data_completeness × 0.3, 
                                      model_certainty × 0.5,
                                      historical_match × 0.2)
```

### 3.3 Rolling Window Analysis
- MAPE is computed over a **rolling 30-day window** using completed orders.
- Latency is computed over a **rolling 24-hour window** for P50/P95/P99.
- Alerting is based on consecutive-day threshold breaches (not single-day spikes).
